## 🎥 The Elevator Pitch

Most automation software asks:

> **"What do you want to automate?"**

FlowSense asks:

> **"Why are you still doing this manually?"**

Instead of making you spend an hour creating automations, FlowSense quietly watches your **high-level desktop activity** *(with permission, calm down FBI 👀)* and discovers repetitive workflows on its own.

When it finds one, it simply asks:

```text
Chrome
   ↓
VS Code
   ↓
ChatGPT
   ↓
Terminal

Seen 23 times this week.

[✨ Create Workflow]
```

# 🔒 Privacy

FlowSense is **local-first**.

Your activity stays on your computer.

The AI never receives:

- ❌ Keystrokes
- ❌ Clipboard
- ❌ Passwords
- ❌ Personal files
- ❌ Browser history

If AI is enabled, it only receives a tiny normalized workflow summary.

Basically:

> "Chrome → VS Code → Terminal"

Not:

> "Here's my life story."
